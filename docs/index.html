erdos97 interactive notes

This site hosts lightweight visual companions for the erdos97 research workspace. These are explanatory aids, not standalone proof certificates.

The first demo shows the small-case obstruction behind the octagon proof note: count isosceles triples, compare base capacities, and watch the octagon equality case run into the exterior-turn budget.

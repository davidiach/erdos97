# Literature-risk notes

The current baseline is intentionally modest. The public Erdős Problems page and the Formal Conjectures repository record this as an open 4-neighbor problem and separately record the known 3-neighbor examples.

Known nearby facts to track:

- Danzer: 9-point convex polygon where every vertex has 3 equidistant vertices, with radius depending on the center.
- Fishburn--Reeds: 20-point convex polygon where every vertex has 3 equidistant vertices at a common radius.

These do not imply a 4-neighbor construction.

Before claiming a solution, do a broader literature sweep around:

- repeated distances in convex polygons;
- Erdős--Moser type convex polygon distance problems;
- Danzer configurations;
- Fishburn--Reeds unit-distance configurations;
- distance multiplicities and convex position;
- oriented-matroid realizability with metric constraints.

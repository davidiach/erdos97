# Scalable Strict-Cycle Bridge Control

Status: `EXACT_ABSTRACT_NEGATIVE_CONTROL`.

This note gives an infinite family of singleton-rich selected-row systems
satisfying the currently isolated incidence, fragile-cover, good-deletion,
hinge, and turn conditions.  It has no Kalmanson self-edge, while its shortest
vertex-circle strict cycle has unbounded length.  The family is not a
Euclidean realization and is not a counterexample to Erdos Problem #97.

## Family

For an integer `k >= 8`, put

```text
n = 6k - 1,
D = {k+1, 2k+3, 3k+1, 5k} in Z/n,
S_i = i + D.
```

Write the four offsets as `A<B<C<E`.  Interpret `S_i` abstractly as the only
rich class at center `i`, declared to be an exact four-tie.  This supplies the
full-rich-class bookkeeping used below; it does not supply coordinates or a
circle realization.

## Minimality shadows

The selected digraph is the Cayley digraph of `Z/n` generated by `D`.  It is
strongly connected, because

```text
B - 2A = (2k+3) - 2(k+1) = 1.
```

In a finite group the directed semigroup generated by `A,B` is their additive
subgroup (a positive multiple supplies each inverse), so this identity makes
every residue reachable from every other residue.

Consequently no nonempty proper set `Q` is row-closed: if `S_i` is contained
in `Q` for every `i in Q`, then no selected arc leaves `Q`, contrary to strong
connectivity.

This is exactly the singleton-rich good-deletion shadow of cardinal
minimality.  From any nonempty proper deleted seed `H`, let `Q` be the
surviving labels.  If every surviving center retained all four witnesses,
`Q` would be a nonempty proper row-closed set.  Hence some surviving center is
good.  Deleting it and repeating gives a good-deletion order from every
nonempty seed.

The map

```text
pi(x) = x - A
```

is a bijective essential fragile-cover assignment because `x in S_{pi(x)}`.
Every exact critical row is used, and the rows cover all labels.  Thus the
family satisfies the combinatorial consequences isolated in
`docs/minimal-fragile-cover-bridge.md`, without claiming that those
consequences characterize a minimal geometric counterexample.

Strong connectivity also shows that this family has no proper full
selected-row subsystem, in particular no closed subsystem on `9` or `10`
labels.

## Pair cap and crossing

The six positive offset differences are

```text
B-A = k+2,       C-A = 2k,       E-A = 4k-1,
C-B = k-2,       E-B = 3k-3,     E-C = 2k-1.
```

Their complements modulo `n` are

```text
5k-3, 4k-1, 2k, 5k+1, 3k+2, 4k.
```

For `k >= 8`, the only repetitions are at the opposite shifts

```text
s = 2k,
-s = 4k-1.
```

Indeed, after suppressing those two repeated occurrences, the values have the
strict order

```text
k-2 < k+2 < 2k-1 < 2k < 3k-3 < 3k+2
    < 4k-1 < 4k < 5k-3 < 5k+1.
```

Therefore two rows intersect in at most two witnesses and a witness pair
occurs in at most two rows.  At shift `s`, rows `S_i` and `S_{i+s}` share

```text
{i+A, i+C} = {i+k+1, i+3k+1}.
```

Starting cyclically at `i`, the four relevant labels occur as

```text
i < i+A < i+s < i+C.
```

Thus the center chord and common-witness chord cross.  Reversing the two rows
handles shift `-s`, so every two-overlap obeys the radical-axis crossing rule.

## No reciprocal pair and no hinge

Directly from the four formulas, no two offsets sum to `n` when `k >= 8`.
Hence

```text
D cap (-D) = empty.
```

Every selected unordered pair belongs to exactly one row-radius class, so the
`n` selected radius classes are disjoint four-spoke stars.  In particular
there is no reciprocal selected pair.

An equilateral hinge requires two of its centers to select each other: its
rows at `a` and `b` require both `b in S_a` and `a in S_b`.  The empty
intersection above therefore proves that the family is hinge-free.

## No Kalmanson self-edge

There is also no strict Kalmanson row that reduces to zero in the full
selected-distance quotient.  The following slightly more general observation
explains why.

Suppose selected radius classes are disjoint stars, row intersections have
size at most two, and the two-overlap crossing rule holds.  For cyclic labels
`a<b<c<d`, a Kalmanson self-edge would have to match the two positive pairs
`ac,bd` to the two negative pairs.  Equivalent distinct pairs in a star share
its center.

For K1, whose negative pairs are `ab,cd`, the two possible matchings force
either rows `a,d` to share `{b,c}`, or rows `b,c` to share `{a,d}`.  In both
cases the center and witness chords do not cross.  For K2, whose negative
pairs are `ad,bc`, the matchings similarly force rows `a,b` to share `{c,d}`
or rows `c,d` to share `{a,b}`.  Again the chords do not cross.  Every case
contradicts the crossing rule.

Thus the no-reciprocal and crossing properties give a symbolic proof of zero
Kalmanson self-edges.  The replay independently enumerates all
`2*binom(n,4)` strict Kalmanson rows and obtains the same zero count.

## No primitive Kalmanson inverse pair

The same family also escapes the two-inequality positive-scalar inverse filter for
every `k >= 8`.

First note that every nonzero reduced Kalmanson coefficient is `+1` or `-1`.
The two positive pairs in a strict Kalmanson row are disjoint, as are the two
negative pairs.  A selected quotient class is a star, so it cannot contain
two same-sign pairs; a nonselected class is one literal pair and cannot do so
either.  The zero vector was excluded in the preceding section.  Hence every
reduced row is already primitive.  If two such integer vectors are negative
positive-scalar multiples, the scalar must be `1`.

There is a compact exact all-parameter decision for the remaining inverse
condition.  Translate the first cyclic quadruple to

```text
0 < b < c < d < n
```

and write the second as `0 <= e < f < g < h < n`.  A cyclic rotation can
exchange K1 and K2, but both kinds are included.  Across two strict rows there
are four positive and four negative pair occurrences.  Their quotient vectors
sum to zero exactly when the positive occurrences can be bijected to the
negative occurrences in the same quotient classes.  This gives

```text
2 * 2 * 4! = 96
```

matching templates.

For `k >= 6` the radius classes are disjoint stars.  Two pair occurrences
`{x,y}` and `{u,v}` then have the same quotient class exactly when they are
the same unordered pair, or they share a center that selects both other
endpoints.  Directed selection has the linear encoding

```text
Sel(x,y) iff y-x is one of D or one of D-n.
```

Thus every matching template is quantifier-free linear integer arithmetic in
`k,b,c,d,e,f,g,h`.  The exact SMT replay gives:

| row kinds | impossible | only k=6 | only k=7 | k=6 or 7 | other/unknown |
|---|---:|---:|---:|---:|---:|
| K1 / K1 | 18 | 1 | 4 | 1 | 0 |
| K1 / K2 | 17 | 4 | 2 | 1 | 0 |
| K2 / K1 | 17 | 2 | 4 | 1 | 0 |
| K2 / K2 | 18 | 4 | 1 | 1 | 0 |
| total | 70 | 11 | 11 | 4 | 0 |

In particular, the disjunction of all 96 templates is `UNSAT` for `k >= 8`.
The exception controls are genuine.  At `k=6`, the reduced rows

```text
K2(0,4,20,24) = R_24 - P_{0,24},
K2(0,8,24,28) = P_{0,24} - R_24
```

are inverse.  At `k=7`, the reduced rows

```text
K1(0,8,26,32) = P_{0,26} - R_0,
K2(0,9,17,26) = R_0 - P_{0,26}
```

are inverse.  Here `R_i` is the selected radius class at center `i`, while
`P_{a,b}` is a singleton nonselected pair class.  The replay finds `SAT` at
both controls and proves `UNSAT` for every `k >= 6` other than `6,7`.

For an independent finite check, the main report canonically enumerates
primitive quotient vectors modulo translation.  It finds `7,580` vector
orbits and zero inverse-orbit pairs at `k=8`, and `11,040` vector orbits and
zero inverse-orbit pairs at `k=9`.

## Strict turn feasibility

Use the uniform normalized exterior-turn assignment

```text
t_j = 4/n.
```

For a tied offset pair `a<b`, the turn-inequality lemma uses a forward support
of size `b-1` and a reverse support of size `n-a-1`.  Across the six pairs in
`D`, the minima are

```text
minimum forward support = B-1   = 2k+2,
minimum reverse support = n-C-1 = 3k-3.
```

Both inequalities are strict:

```text
4(2k+2) > 6k-1,
4(3k-3) > 6k-1.
```

So the family lies strictly inside the current turn relaxation; it is not a
boundary artifact of the stored weak inequalities.

## The first vertex-circle cycle has length n

The selected-spoke unoriented gap types are

```text
k-1, k+1, 2k+3, 3k-2.
```

The five witness-chord gap types are

```text
AB = k+2,
AC and AE = 2k,
BC = k-2,
BE = 3k-3,
CE = 2k-1.
```

For `k >= 8`, the latter five are distinct except for `AC/AE`, and they are
disjoint from the selected-spoke gaps.  One direct check is the interleaving

```text
k-2 < k-1 < k+1 < k+2 < 2k-1 < 2k
    < 2k+3 < 3k-3 < 3k-2.
```

Consequently these witness chords remain singleton distance classes unless
they are literally the same unordered pair.

Let

```text
X_i = AE_i = {i+A, i+E}.
```

The nested-chord inequality in row `i` gives `AE_i > AC_i`.  With `s=2k`,

```text
AC_i = {i+A, i+C} = X_{i+s},
```

so the strict quotient graph contains

```text
X_i -> X_{i+s}.
```

The complete containment table for four ordered witnesses is

```text
AC -> AB, BC,
AE -> AB, AC, BC, BE, CE,
BE -> BC, CE.
```

All arrows other than `AE_i -> AC_i` drain into singleton branches.  Hence
the unique cyclic strongly connected component is the step-`s` cycle on the
`X_i`.  Finally,

```text
gcd(2k, 6k-1) = 1,
```

because the gcd divides `3(2k)-(6k-1)=1`.  The cycle therefore has length
exactly `n`; there is no vertex-circle self-edge or shorter strict cycle.

The exact formulas are

```text
two-overlaps:              n
strict edges:              9n
strict graph nodes:        5n
singleton SCCs:            4n
one cyclic SCC:             n nodes and n internal edges
Kalmanson self-edges:       0
primitive inverse orbits:   0
shortest strict cycle:      n
```

The default replay gives this exact table:

## Consequence and scope boundary

The family disproves any implication from the listed abstract axioms to a
Kalmanson self-edge, a primitive two-inequality Kalmanson inverse pair, a
vertex-circle cycle of universally bounded length, or a proper row-closed
`9`/`10`-label localization.  Any such bridge needs either a different
bounded multi-row pattern, global support that may grow with `n`, or an
additional metric ingredient.

This statement is deliberately narrower than a claim about the full
Kalmanson cone.  The construction and replay do **not** rule out bounded
positive circuits using three or more Kalmanson inequalities; in particular,
the three-inequality case is unclassified. In fact the first member, `k=8`,
has the exact four-inequality circuit

```text
d(1,16) > d(16,27) > d(16,37) > d(1,37) > d(1,16).
```

The four arrows reduce respectively from

```text
K1(1,8,16,27),
K2(16,18,27,37),
K2(1,16,23,37),
K1(1,16,37,44),
```

using the selected equalities in rows `8,18,23,44`.  Thus this family is not
a negative control for bounded general Kalmanson certificates.  It also says
nothing about Ptolemy, Gram-rank, or other global Euclidean constraints.

Indeed, Altman's diagonal-sum theorem already rules out this family in the
displayed cyclic order.  Summing the four equal-spoke relations over all
centers would force equality among

```text
U_{k-1}, U_{k+1}, U_{2k+3}, U_{3k-2},
```

contradicting the strict Altman chain because these four chord orders are
distinct.  This growing-support contradiction is exactly the sort of global
ingredient that the bounded vertex-circle screen does not retain.  Thus
passing the abstract tests does not make the family geometrically realizable
and does not refute Erdos Problem #97.

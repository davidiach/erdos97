# C19 Kalmanson Prefix Branch Pilot

Status: `EXACT_OBSTRUCTION` for 100 sampled three-boundary-prefix branches of
the fixed selected-witness pattern `C19_skew`.

This is a bounded pilot, not an all-order C19 search. It does not prove Erdos
Problem #97 and does not kill the abstract `C19_skew` pattern across all cyclic
orders.

## Method

The script fixes label `0` as the cyclic-order anchor and samples the first 128
reflection-pruned three-boundary-prefix states generated by deterministic DFS.
Each state represents orders of the form:

```text
0, left[0], left[1], left[2], middle permutation,
right[2], right[1], right[0]
```

For each sampled prefix, the script keeps only Kalmanson quadrilateral
inequalities whose four labels have a unique order in every completion of the
prefix. Any quadrilateral containing two or more unordered middle labels is
discarded. With `n = 19` and three boundary pairs, each sampled branch has 910
prefix-forced Kalmanson rows.

If a positive-integer Farkas combination of these rows has zero total
coefficient after quotienting by the selected-distance equalities, the
contradiction `0 > 0` holds for every completion of that one sampled prefix.

## Claim Boundary

Each closed sampled branch is an exact obstruction for every cyclic-order
completion of that fixed three-boundary-prefix state. The 28 unclosed sampled
branches are not counterexamples and are not evidence of realizability.

This pilot samples only 128 of 6,683,040 canonical three-boundary-prefix states.
It is useful evidence that the C13 prefix-forced Kalmanson machinery transfers
to `C19_skew`, but it is not an exhaustive obstruction.
